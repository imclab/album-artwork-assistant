NSURL(NDCarbonUtilitiesInfoFlags)


Abstract

Adds methods to NSURL

Discussion

Adds methods to simplify testing of the flags returned from finderInfoFlags:type:creator:

Methods

hasCustomIcon

Abstract: Test if a file has a custom icon. 
- (BOOL)hasCustomIcon;

Test to see if the file refered to by the receiver has a custom icon. The is equivelent to testing for the kHasCustomIcon flag

Result: Returns YES if the file has a custom icon.
