Methods


descriptorWithAEDesc:

Abstract: Returns a NSAppleEventDescriptor for the AEDesc. 
+ (id)descriptorWithAEDesc:(const AEDesc *)aeDesc;

Allocates, initializes and returns an NSAppleEventDescriptor that copies the Carbon AEDesc structure pointed to by aeDesc. Returns nil if an error occurs.

Parameters

NameDescription
aeDescA Carbon AEDesc structure
Result: A NSAppleEventDescriptor

descriptorWithAEDescNoCopy:

Abstract: Returns a NSAppleEventDescriptor for the AEDesc. 
+ (id)descriptorWithAEDescNoCopy:(const AEDesc *)aeDesc;

Allocates, initializes and returns an NSAppleEventDescriptor that takes ownership of the Carbon AEDesc structure pointed to by aeDesc. Returns nil if an error occurs. The initialized object takes responsibility for calling the AEDisposeDesc function on the AEDesc at object deallocation time.

Parameters

NameDescription
aeDescA Carbon AEDesc structure.
Result: A NSAppleEventDescriptor

isTargetCurrentProcess

Abstract: Determines if target is current process. 
- (BOOL)isTargetCurrentProcess;

If the recevier is a AppleEvent that contains a target ProcessSerialNumber that is the current process (ie you application) then this method returns YES.

Result: Returns YES if the recevier is an AppleEvent for the current process.
