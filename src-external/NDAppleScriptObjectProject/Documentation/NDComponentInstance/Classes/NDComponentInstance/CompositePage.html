NDComponentInstance


Abstract

A class to represent a component instance.

Discussion

A component instance is a connection to a component (osa component) used to compile and execute AppleScripts. This class is to be used with NDAppleScriptObject.

Methods

activeTarget

Abstract: returns the active target as set by the method setActiveTarget: 
- (id)activeTarget;

See the method setActiveTarget: for a discussion.

appleEventSendTarget

Abstract: returns the object that handles any AppleEvents 
- (id)appleEventSendTarget;

See the method setAppleEventSendTarget: for a discussion.

Result: the AppleEvent send target.

closeSharedComponentInstance

Abstract: Close the shared component instance. 
+ (void)closeSharedComponentInstance;

This can be called before you application closes or when it has finished with the shared component instance.

componentInstance

Abstract: Allocates and initializes a NDComponentInstance 
+ (id)componentInstance;

Returns a NDComponentInstance with a connection to the default OSA Apple component.

Result: A NDComponentInstance

componentInstanceWithComponent:

Abstract: Allocates and initializes a NDComponentInstance 
+ (id)componentInstanceWithComponent:(Component)component;

Returns a NDComponentInstance with a connection to the supplied component.

Parameters

NameDescription
componentA component, if NULL then the default AppleScript component is used..
Result: A NDComponentInstance

findNextComponent:

Abstract: Finds the next OSA component. 
+ (Component)findNextComponent;

Can be used by init methods that take a component parameter so that a script can be connected to it own OSA component. This is useful if you want to execute AppleScripts within separate threads as each OSA component is not thread safe.

Result: Returns a OSA component.

init

Abstract: Intializes a NDComponentInstance. 
- (id)init;

The receiver opens a connection with the default AppleScript component.

Result: The initialized NDComponentInstance

initWithComponent:

Abstract: Intialize a NDComponentInstance. 
- (id)initWithComponent:(Component)component;

The receiver opens a connection with the supplied component.

Parameters

NameDescription
componentA component, if NULL then the default AppleScript component is used..
Result: The initialized NDComponentInstance

setActiveTarget:

Abstract: sets the object which is periodicly sent the message appleScriptActive:. 
- (void)setActiveTarget:(id)target;

will the script is running it will periodocly give up time for you to do some other processing.

Parameters

NameDescription
targetAn object that implements the protocol NDAppleScriptObjectActive.

setAppleEventSendTarget:.

Abstract: sets the object that any handles any AppleEvent the script atempts to send. 
- (void)setAppleEventSendTarget:(id)target;

If the send target is set any AppleEvents are sent to the send target to be processed, otherwise NDComponentInstance will handle the event itself by utilising the OSA default send procedure. One use of this is when executing a script in thread any AppleEvents that require user interaction sent to the current procees need to be sent from the main thread.

Parameters

NameDescription
targetAn object that implements the protocol NDAppleScriptObjectSendEvent.

setDefaultTarget:

Abstract: sets the default target for any AppleEvents. 
- (void)setDefaultTarget:(NSAppleEventDescriptor *)defaultTarget;

any AppleEvents not enclosed in a tell statement by default go to the current process (your application). With this method you can provide a different default target.

Parameters

NameDescription
defaultTargetan NSAppleEventDescriptor containing the target descriptor

setDefaultTargetAsCreator:

Abstract: sets the default target, specified by creator code, for any AppleEvents 
- (void)setDefaultTargetAsCreator:(OSType)creator;

same as setDefaultTarget: but passing the creator code of an application to specify the target process.

Parameters

NameDescription
creatoran OSType creator code of the processes application.

setFinderAsDefaultTarget.

Abstract: sets the default target as Finder for any AppleEvents 
- (void)setFinderAsDefaultTarget;

passes the Finders creator code to setDefaultTargetAsCreator:.

sharedComponentInstance

Abstract: Returns the shared component instance. 
+ (id)sharedComponentInstance;

This is the single component all NDAppleScriptObjects use if you do not give it a NDComponentInstance. The shared component instance has a connection with the default OSA Apple component.

Result: Returns NDComponentInstance component.
