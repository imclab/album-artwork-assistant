NSMutableString(NDUtilities)


Abstract

Addition methods for NSString

Discussion

Additonal useful methods for NSMutableString, that can be used in general situations.

Methods

prependFormat:

Abstract: Prepend a string format. 
- (void)prependFormat:(NSString *)format, ...;

Adds a constructed string to the beginning of the receiver. Creates the new string using NSString's stringWithFormat: method with the arguments listed. Raises an NSInvalidArgumentException if format is nil.

Parameters

NameDescription
formatThe format string to append, followed by values to insert.

prependString:

Abstract: Prepend a string. 
- (void)prependString:(NSString *)string;

Adds the characters of string to the beginning of the receiver. string may not be nil.

Parameters

NameDescription
stringThe string to append.
