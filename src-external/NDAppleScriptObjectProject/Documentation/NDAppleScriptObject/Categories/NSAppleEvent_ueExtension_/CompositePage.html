NSAppleEventDescriptor(NDAppleScriptValueExtension)


Abstract

category of NSAppleEventDescriptor.

Discussion

adds a method to NSAppleEventDescriptor retrieve a NDAppleScriptObject from a NSAppleEventDescriptor.

Methods

appleScriptValue

Abstract: Category method for NSAppleEventDescriptor (NDAppleScriptObjectValueExtension), converts any script data within a AppleEvent descriptor into an NDAppleScriptObject 
- (NDAppleScriptObject *)appleScriptValue;

If an AppleScript return a AppleScript as it’s result, this method can be used to convert the result NSAppleEventDescriptor into a NDAppleScriptObject. The NSAppleEventDescriptor method objectValue will use this method if available.

Result: a NDAppleScriptObject object for the AppleScript contained within the AppleEvent descriptor.
