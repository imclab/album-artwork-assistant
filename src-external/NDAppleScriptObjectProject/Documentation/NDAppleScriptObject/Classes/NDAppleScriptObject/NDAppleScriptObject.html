NDAppleScriptObject

Abstract: Class to represent an AppleScript.

As well as representing an AppleScript, NDAppleScriptObject also can maintain seperate context for each AppleScript, useful if you want to run each script within a seperate thread. NDAppleScriptObject is interface compatabile with NSAppleScript
