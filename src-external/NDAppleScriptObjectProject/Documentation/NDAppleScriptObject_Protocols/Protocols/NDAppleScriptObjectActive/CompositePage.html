NDAppleScriptObjectActive


Abstract

Protocol used by NDAppleScriptObject

Discussion

The protocol NDAppleScriptObjectActive is implemented by any object that wishes to.

Methods

appleScriptActive

Abstract: Method called periodical during script execution. 
- (BOOL)appleScriptActive;

Allows you to do some processing durring execution of a AppleScript with the same thread.
